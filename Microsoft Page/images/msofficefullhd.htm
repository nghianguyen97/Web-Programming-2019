Microsoft Office — application software office package. It has been created by Microsoft company for Microsoft Windows and Apple Mac OS X operation systems. This program package includes softwares for work with different documents: text processor, spreadsheet, database, etc. Microsoft Office OLE is a server for objects. It supports macros written in Visual Basic language.

Microsoft Office programs

  • Microsoft Office Word — text processor. It is used to prepare and edit different documents.
  • Microsoft Office Excel — table processor.
  • Microsoft Office Outlook — personal communicator. The program includes: calendar, tasks plan, e-mail, and address manual.
  • Microsoft Office PowerPoint — is used for preparing presentations.
  • Microsoft Office Access — database management system. It is an application software for creating DB and managing it.
  • Microsoft Office InfoPath — application used for collecting information.
  • Microsoft Office Communicator — provides information exchange between people.
  • Microsoft Office Publisher — is used for publishing.
